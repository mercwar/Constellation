# 🔧 Installation

## 1️⃣ Download the Browser  
Download the ZIP archive from this repository and extract it anywhere on your system.

## 2️⃣ Configure `config.json`  
Update the file with your GitHub information:

```json
{
  "owner": "your-github-username",
  "repo": "your-default-repo",
  "branch": "main",
  "api_base": "https://api.github.com/repos",
  "raw_base": "https://raw.githubusercontent.com",
  "pages_base": "https://your-github-username.github.io"
}
```

### Required Fields

| Field | Description |
|-------|-------------|
| `owner` | Your GitHub username |
| `repo` | Default repository |
| `branch` | Typically `main` |
| `api_base` | Must remain `https://api.github.com/repos` |
| `raw_base` | Must remain `https://raw.githubusercontent.com` |
| `pages_base` | Your GitHub Pages root URL |

---

## 3️⃣ Upload to GitHub  
1. Create a new GitHub repository  
2. Upload all extracted files  
3. Commit and push  

---

## 4️⃣ Enable GitHub Pages  
1. Go to **Settings → Pages**  
2. Under **Build and deployment**, select:  
   - **Source:** Deploy from branch  
   - **Branch:** `main`  
   - **Folder:** `/ (root)`  
3. Save your settings  

Your browser will be available at:

```
https://your-username.github.io/<your_repo_name>/
```

# 🚀 Usage

To browse any public GitHub repository, append:

```
?index=<repository>
```

### Example

```
https://your-username.github.io/Constellation/LIVE/index.html?index=Cyborg
```
You only need to serve the browser from one repo
If no `index` parameter is provided, the browser loads the default repository defined in `config.json`.
