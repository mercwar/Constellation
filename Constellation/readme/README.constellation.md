# MERCWAR CONSTELLATION OVERVIEW

The Mercwar Constellation is a structured map of all repos used by
Sentinel, Nexus, and the AVIS-Datalake.

## REPOS
- AVIS (Core)
- CYBORG (Engine)
- FIRE-GEM (UI)
- ROBO-KNIGHT (Player)
- DARKSTAR (System)
- BROWSERX (Browser)
- SENTINEL (Robot)
- NEXUS (Robot)

## PURPOSE
Provide a unified star-map for:
- Navigation (Sentinel)
- Semantic binding (Nexus)
- Ingestion (Walker)
