# MERCWAR CONSTELLATION — ROOT README

The Constellation is the unified intelligence map for all Mercwar
repositories. It defines how robots (Sentinel, Nexus, Walker) navigate,
ingest, bind, and evolve intelligence through the AVIS-Datalake.

## CORE COMPONENTS
- /maps — Constellation maps + robot rules
- /robots — Sentinel, Nexus, Walker logic
- /sources — Repo mirrors for ingestion
- /queries — Robot query definitions
- /tools — Sync, verify, and build utilities
- /readme — Secondary documentation set

## ROBOT INTELLIGENCE LOOP
1. Sentinel scans constellation
2. Walker extracts units
3. AVIS normalizes data
4. Nexus binds knowledge
5. Models stored in datalake
6. Sentinel upgrades intelligence
7. Loop repeats

## PURPOSE
Provide a stackable intelligence substrate for robots.
